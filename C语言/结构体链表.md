###  结构体创建
结构体有两种创建方式，一个是打上标签，一个是利用typedef 取得和标签一样的效果，实际使用则是更加像类型名。
1. 标签
```c
struct TIME{
	int hour;
	float min;
	double sec;
};
struct TIME x;
struct TIME y[10],*z;
```
2. 类型名
```c
typedef struct {
	int hour;
	float min;
	double sec;
}TIME;
TIME x;
TIME y[10],*z;
```
### 结构体访问
两种访问方式
直接 
struct TIME time;
time.sa[4].c
间接
指向结构体的数组 可以直接用操作符 -> 访问内部变量

### 结构体存储
边界对齐：
1. 首地址的存储地址必须被4整除
2. 如果声明了相同类型的第二个变量它的存储位置也必须满足4这个边界条件

### 作为函数参数传递
尽量使用指针进行地址传递，这样子一般会有更高的效率
