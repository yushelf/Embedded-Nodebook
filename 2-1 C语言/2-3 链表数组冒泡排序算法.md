## 冒泡排序
核心思想：for只是用来计数的，想象每一个都往下沉，所以每次循环必须回到起点，每循环一次，次数就要少一次。
`链表版本`
```c
void soert_list(PNODE pHead)
{
	int i,j,length;
	int tmp;
	PNODE ptr = pHead->pNext;
	length = length_list(pHead);
	for(i = 0;i < length;i++)
	{
		for(j = i;j < length-1;j++)//少比较一个
		{
			if(ptr->data < ptr->pNext->data)
			{
				tmp = ptr->pNext->data;
				ptr->pNext->data = ptr->data;
				ptr->data = tmp; 
			}		
			ptr = ptr->pNext;	
		}
		ptr = pHead->pNext;//指针回头
	}
}
```
`数组版本`

```c
#include <stdio.h>
#include <stdlib.h>

int main(void)
{
	int i,j;
	int tmp;
	int val;
	int len;
	int num[10] = {0};
	printf("请输入数组长度：\n");
	scanf("%d",&len);
	for(i = 0;i<len;i++)
	{	
		scanf("%d",&num[i]);
	}
	for(i = 0;i<len;i++)
	{
		for(j = 0;j<len-i-1;j++)
		{
			if(num[j]>num[j+1])
			{
				tmp = num[j+1];
				num[j+1] = num[j];
				num[j] = tmp;
			}
		}
	}
	for(i = 0;i<len;i++)
	{
		printf("%d ",num[i]);
	}
```
